NarrateX - Local text-to-speech audiobook engine

NarrateX is a locally-run text-to-speech desktop application that converts ebooks and documents into real-time narrated audio. Designed for long-form audiobook listening without relying on cloud services, it provides offline TTS with streaming playback and a layered architecture ensuring deterministic behaviour and reliability.

Local. Deterministic. No cloud dependency.
NarrateX supports EPUB, PDF and plain text files, allowing users to convert books into narrated audio in real time. Unlike cloud-based text-to-speech services, NarrateX runs entirely locally, giving full control over performance, privacy and reliability.
